# error code

1:成功

1XX: master 错误

2XX: worker 错误

3XX : 推流端错误

4XX: 逻辑错误



201:下载JSON文件失败

202：读本地JSON文件失败

401: 课程已经预约（重复预约）

402: 课程未预约

403: 命令错误

404:课程未预约或者课程正在上课中，不能做取消预约动作

405：没有上课，不能下课

500: 未知错误
